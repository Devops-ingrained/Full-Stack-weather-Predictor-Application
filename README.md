# 🌦️ Predictive Weather & Business Strategy Dashboard

A full-stack data analytics and strategic planning application built with **Streamlit**. This platform processes historical climate data, utilizes Machine Learning models to forecast future weather trends, and integrates a Generative AI Copilot to deliver executive business risk reports.

---

## 🚀 System Capabilities

* **Automated Data Engineering:** Securely pulls and caches multi-variable climate data.
* **Predictive ML Pipeline:** Evaluates baseline trends to run real-time inference calculations using customized regression models.
* **GenAI Strategic Copilot:** Uses advanced language models via the Groq API gateway to provide corporate risk analysis for logistics and infrastructure.
* **Performance Architecture:** Employs state management to decouple machine learning processing from LLM network handling for maximum execution speed.
